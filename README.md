# Создание нового массива из уже существующего.

**Задание:**
___
> Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа.

<br>

**Описание решения:**
___
1. Создаем массив **initialArr** размером 5 элементов.

2. С помощью метода **FillArray** просим пользователя ввести 5 строк и записываем данные в массив **initialArr**.
    > Метод **FillArray**
    1. На вход принимается массив.
    2. С помощью цикла записываем введенные пользователем данные в данный массив.
    3. Возвращаем данный массив.

<br>

3. Создаем новый массив **finalArr** с размером, который получаем из **initialArr** с помощью метода **FindArrSize**.

    > Метод **FindArrSize**
    1. На вход принимается массив.
    2. Задаем переменную **size**
    3. С помощью цикла проверяем сколько элементов имеют длину, которая меньше или равна 3.
    4. Если условие выполняется, добавляем 1 к переменной **size**.
    5. После завершения цикла, возвращаем **size**.

<br>

4. Заполняем массив **finalArr** с помощью метода **FillNewArr**

    >Метод **FillNewArr**
    1. На вход принимаются 2 массива. **initialArr** - заполненный пользователем массив. **finalArr** - новый массив.

    2. Задаем переменные **i** (индекс) и **count**.

    3. Создаем цикл. 

    4. Если **i** меньше длины массива **initialArr**, цикл продолжается. Если условие не выполняется, выводим **finalArr**.

    5. Берем элемент массива с индексом **i** и проверяем является ли его длина меньшей или равной 3. 
        * Если условие выполняется, записываем значение элемента **initialArr[i]** в **finalArr[count]**. Прибавляем 1 к переменным **count** и **i**, возвращаемся к пункту 4.
        * Если условие не выполняется, добавляем 1 к переменной **i**, возвращаемся к пункту 4.

___
