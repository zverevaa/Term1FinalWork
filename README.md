# Создание нового массива из уже существующего.

**Задание:**
> Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа.

<br>

**Описание решения:**
___
1. На вход принимаются 2 массива. **initialArr** - заполненный массив. **finalArr** - новый массив.

2. Задаем переменные **i** (индекс) и **count**.

3. Создаем цикл. 

4. Если **i** меньше длины массива **initialArr**, цикл продолжается. Если условие не выполняется, выводим **finalArr**.

5. Берем элемент массива с индексом **i** и проверяем является ли его длина меньшей или равной 3. 
    * Если условие выполняется, записываем значение элемента **initialArr[i]** в **finalArr[count]**. Прибавляем 1 к переменным **count** и **i**, возвращаемся к пункту 4.
    * Если условие не выполняется, добавляем 1 к переменной **i**, возвращаемся к пункту 4.

___

Задача решена на языке программирования C# в файле [Program.cs]. Блок-схема находится в файле [diagram.drawio](/diagram.drawio).
